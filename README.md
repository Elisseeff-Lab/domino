# `Domino`

<a name="archive-notice"></a>
> [!WARNING]
> This repository is archived and no longer maintained. It is preserved for reproducibility purposes. For ongoing development, please see [dominoSignal](https://fertiglab.github.io/dominoSignal/).

## Accessing the Original `Domino` Package
The `Domino` R package was published in [Computational reconstruction of the signaling networks surrounding implanted biomaterials from single-cell transcriptomics](https://doi.org/10.1038/s41551-021-00770-5) in Nature Biomedical Engineering in 2021.

Code used in the manuscript has been uploaded to [Zenodo](https://doi.org/10.5281/zenodo.8277233) and is also available as github release [domino v1.0.0](https://github.com/Elisseeff-Lab/domino/releases/tag/v1.0.0). The original documentation for `Domino` is available in the [original-README.md file](https://github.com/Elisseeff-Lab/domino/blob/master/original-README.md).

## Introducing `dominoSignal` <a href="https://fertiglab.github.io/dominoSignal/"><img src="man/figures/logo.svg" align="right" height="100" alt="dominoSignal hex logo" /></a>

Active development now takes place in the `dominoSignal` package, which is available for installation on Bioconductor.

- Bioconductor stable release: [https://bioconductor.org/packages/release/bioc/html/dominoSignal.html](https://bioconductor.org/packages/release/bioc/html/dominoSignal.html)
- Documentation website: [https://FertigLab.github.io/dominoSignal/](https://fertiglab.github.io/dominoSignal/)
- GitHub repository: [https://github.com/FertigLab/dominoSignal](https://github.com/FertigLab/dominoSignal)

Please note that while `dominoSignal` is based on the `Domino` package, package workflows and functionality have evolved, such that the `dominoSignal` package might not be a drop in replacement for all use cases.
